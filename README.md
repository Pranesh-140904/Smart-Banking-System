# 🏦 Smart Banking System

## 📖 Overview

Smart Banking System is a full-stack web application developed using Python, Django, HTML, CSS, JavaScript, and SQLite. The system simulates modern banking operations by allowing users to manage accounts, perform transactions, track balances, and view banking activities through a secure and user-friendly interface.

This project demonstrates real-world banking functionalities and showcases full-stack development skills using Django.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Pranesh-140904/Smart-Banking-System.git
```

### Move to Project Directory

```bash
cd Smart-Banking-System
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

Windows:

```bash
venv\Scripts\activate
```

Linux/Mac:

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Migrations

```bash
python manage.py migrate
```

### Start Server

```bash
python manage.py runserver
```

### Open Browser

```text
http://127.0.0.1:8000/
```
